Данный бот выполняет парсинг на сервера nasa и отправляет ежедневные картинки космоса, последние снимки Земли,
снимки Марса.

Для запуска бота в каталоге с файлом main.py должны находится файлы api_tools.py, config.json, bot.py,
joined.txt(будет заменено на базу данных sql), keyboard.py, project_tools.py.
Запуск производится от main.py

main.py - файл запускающий бота
имеет следующие функции:
main - основная функция файла, запускающая бота
bot.py - файл с классом бота TelegramBot
имеет следующие классы и функции:
TelegramBot - Класс собравший все функции-обработчики телеграм бота.
из них:
help - функция обработчик команды /help
start - функция обработчик команды /start
dialog - функция отвечающая пользователю на неопознанный запрос
send_apod - функция обработчик команды /apod осуществляющая отправку пользователю ежедневную космическую картинку
send_epic - функция обработчик команды /epic осуществляющая отправку пользователю последние снимки Земли из космоса
send_mars_rover_photos - Функция обработчик команды /mars_rover_photos отправляющая пользователю случайные фото
марса из последних сделанных марсоходами

project_tools.py - Файл содержащий в себе необходимые функции для работы технической составляющей бота
в них входят:
get_response - Функция задающая get запрос серверам api
get_config - Функция получающая информацию из файла конфигурации бота
translator_e_to_r - Функция перевода с Английского на Русский(Будет доработано в скором времени)
joined - Функция регистрации пользователя в боте(Будет использовано для рассылки)
get_joined_users_id - Функция для получения информации о зарегестрированых пользователях

api_tools.py - Файл содержащий в себе необходимые функции для работы бота с api
в них входят:
apod - Функция для получения ежедневной картинки и информации о ней
epic - Функиця для получения последней фотографии Земли из космоса
mars_rover_photos - Функция для получения фотографий с поверхности Марса

joined.txt - Файл с информацией о пользователях, зарегестрированых в боте

keyboard.py - Файл реализующий работу клавиатуры для быстрого набора команд в чате с ботом
